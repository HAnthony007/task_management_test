## 🔧 Installation & Configuration

### 📌 Prérequis

✔️ Bun

### 🚀 Installation
```sh
# Cloner le projet
git clone https://github.com/HAnthony007/task_management_test
cd task_management_test

# Installer les dépendances
bun install

# Lancer le en dévelopement
bun dev

# Builder l'app
bun build

# Lancer
bun start
```
